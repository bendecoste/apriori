Apriori

Based off of the algorithm outlined here: http://en.wikipedia.org/wiki/Apriori_algorithm

To build:
  `make`

to clean:
  `make clean`

Assumes that you have gcc with openmp directives installed.

to run:
  `./main <threshold> <num transactions>`

  eg, `./main 3 10`

what threshold and num transactions are are outlined in the pdf included in this
directory.
